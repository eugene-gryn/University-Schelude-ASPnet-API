### **Идея проекта:**
#### Приложение для уведомлений студента о парах.

### **Описание:**
#### Цель этого приложения уведомлять студентов до начала пары, та уточнять её домашнее задание и указывать его приоритет.

### **Функционал движка:**
	- Профиль пользователя
		- Аватар
		- Имя
		- Логин
		- Пароль
		- Настройки
	- Управление рассписанием
		- Excel загрузка
		- PDF загрузка
		- Ручной ввод
		- Отображение
		- Ближайшая пара
		- Накладка пар
		- Удаление
		- Изменение
	- Управление уведомленями
		- Виды
		- Управление
		- Настройки
	- Управление домашними заданиями
		- Дэдлайн
		- Приоритет
		- Текст
		- Документ
		- Название предмета
